Platform Overview

TradeTrakR and Edgewonk serve different segments of the funded trading community. TradeTrakR emphasizes AI-powered insights with automated rule tracking, while Edgewonk focuses on manual control with deep customization options.

TradeTrakR

Built specifically for funded traders, TradeTrakR combines AI narratives with psychology tracking to help traders stay within evaluation rules. The platform emphasizes automated insights, rule adherence monitoring, and prop-firm compliance features.

Edgewonk

Edgewonk offers desktop-first journaling with extensive customization options. It appeals to traders who prefer manual control over automation, with deep analytics and psychology tracking tools that require more setup but offer greater flexibility.

Data import workflows differ significantly between platforms, affecting daily maintenance overhead.

TradeTrakR Import Workflow

  • Current: Secure CSV imports with validation and error checking
  • Roadmap: Direct sync for Tradovate, NinjaTrader, and Rithmic
  • Prop Integration: Webhook interface for custom prop firm data

Edgewonk Import Workflow

  • Method: Manual CSV templates for NinjaTrader, Sierra Chart, IB
